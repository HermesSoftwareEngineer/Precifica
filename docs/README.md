# API Documentation

This directory contains documentation for the API routes of the Precifica project.

## Route Modules

- [Admin Routes](routes/admin_routes.md) - User management and admin functions.
- [Auth Routes](routes/auth_routes.md) - Authentication (Login, Register, Logout).
- [Bot Routes](routes/bot_routes.md) - AI Chatbot interaction.
- [Conversation Routes](routes/conversation_routes.md) - Chat history and management.
- [Evaluation Routes](routes/evaluation_routes.md) - Property evaluation and comparable listings.
- [Main Routes](routes/main_routes.md) - General application routes.
